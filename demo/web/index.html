ruviz in the browser

This demo uses the new ruviz-web crate to render and interact with plots from WebAssembly. Left-drag pans, right-drag box-zooms, and the mouse wheel zooms around the pointer. The second panel runs through an OffscreenCanvas worker when the browser supports it.

Direct Export

Static export through Plot::render_png_bytes() and Plot::render_to_svg() without creating an interactive canvas session.

Main-thread Canvas

CPU-backed interactive rendering directly on an HTML canvas.

Offscreen Worker

Interactive rendering delegated to a worker using OffscreenCanvas.

Observable Interactive View

Reactive plot data backed by ruviz::data::Observable<Vec<f64>>. Move the phase slider or press Play to mutate the series while keeping the current interactive viewport.
