# ruviz v0.4.3

Released: 2026-04-08

## Highlights

- Notebook widgets now honor `plot.size_px(width, height)` as their display size in notebook outputs, and shrink proportionally when the host cell is narrower than the configured width.
- Notebook widget exports now live behind a right-click menu with `Save PNG` and `Save SVG`, removing the old always-visible buttons while keeping right-drag box zoom available.
- Browser regression coverage now checks notebook-like host sizing, custom export-menu behavior, and secondary-button zoom handling for the blob-backed widget runtime.

## Version Alignment

This release keeps the published workspace packages aligned on one version:

- `ruviz` `0.4.3`

- `ruviz-gpui` `0.4.3`
- npm `ruviz` `0.4.3`
- PyPI `ruviz` `0.4.3`

## Notebook Widget UX

- `plot.size_px(width, height)` now controls both the widget export ratio and the displayed notebook widget size when there is enough room.
- When the notebook output area is narrower than the configured width, the widget clamps to the available width and preserves the same aspect ratio as PNG and SVG exports.
- Export actions are now tucked into a right-click menu so the default notebook widget presentation stays focused on the plot itself.
