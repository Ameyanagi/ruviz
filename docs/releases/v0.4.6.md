# ruviz v0.4.6

Released: 2026-04-08

## Highlights

- Notebook widgets now default to the same `640x480` display size as the exported PNG when `size_px(...)` is not set.
- Notebook outputs now include a bottom-right resize handle so widget size can be adjusted interactively after rendering.
- Holding `Shift` or `Ctrl` while dragging keeps the current aspect ratio, and the right-click export menu now has stronger contrast.

## Version Alignment

This release keeps the published workspace packages aligned on one version:

- `ruviz` `0.4.6`
- `ruviz-web` `0.4.6`
- `ruviz-gpui` `0.4.6`
- npm `ruviz` `0.4.6`
- PyPI `ruviz` `0.4.6`

## Notebook Widget Improvements

- Default notebook widget sizing now matches PNG export defaults instead of stretching to full notebook width.
- Widgets shrink proportionally when the notebook cell is narrower than the natural display size.
- A bottom-right resize handle supports free resize by default and aspect-ratio-locked resize with `Shift` or `Ctrl`.
- Resize pointer handling now covers mouse, touch, and pen input more robustly.
- The custom right-click export menu has improved contrast for notebook hosts with light backgrounds.

## Quality And Validation

This release was locally validated after the `0.4.6` version bump with:

- `cargo check -q`
- `bun run check:web`
- `cd python && uv run cargo test`
- `cd python && uv run python -m pytest`

The notebook widget browser coverage was also re-run locally with:

- `bun run --cwd demo/web test:e2e --grep "python widget bundle"`
