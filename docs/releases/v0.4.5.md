# ruviz v0.4.5

Released: 2026-04-08

## Highlights

- Fixed the `Python Packaging` CI job so it installs the `wasm32-unknown-unknown` target before rebuilding the notebook widget bundle.
- This unblocks the tag-driven GitHub release workflow that failed for `v0.4.4` during the packaging check.
- No intended API changes were made in this release; it is a release-pipeline recovery cut.

## Version Alignment

This release keeps the published workspace packages aligned on one version:

- `ruviz` `0.4.5`
- `ruviz-web` `0.4.5`
- `ruviz-gpui` `0.4.5`
- npm `ruviz` `0.4.5`
- PyPI `ruviz` `0.4.5`

## Release Pipeline Fix

- `.github/workflows/ci.yml` now installs `wasm32-unknown-unknown` in the `Python Packaging` job before `bun run build:python-widget`.
- The underlying packaging and release changes introduced for `0.4.4` remain in place:
  - Linux wheel publishing in release CI
  - slimmer source distributions
  - release workflow recovery runs pinned to tag refs

## Quality And Validation

This release was locally rechecked after the `0.4.5` version bump with:

- `cargo check -q`
- `bun run check:web`
- `cd python && uv run cargo test`
- `cd python && uv run python -m pytest`

The generated sdist was also rebuilt and smoke-installed locally.
