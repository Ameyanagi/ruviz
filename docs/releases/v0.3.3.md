# ruviz v0.3.3

Released: 2026-04-01

## Highlights

- Completed the unified tag-driven release flow across crates.io, npm, GitHub Releases, and PyPI by removing the unsupported plain Linux wheel from the PyPI upload set.
- PyPI now publishes the source distribution plus macOS Intel, macOS Apple Silicon, and Windows wheels for `ruviz`.
- Linux Python installs currently fall back to the source distribution until a compliant `manylinux` or `musllinux` wheel path is added.

## Version Alignment

This release keeps the published workspace packages aligned on one version:

- `ruviz` `0.3.3`
- `ruviz-web` `0.3.3`
- `ruviz-gpui` `0.3.3`
- npm `ruviz` `0.3.3`
- PyPI `ruviz` `0.3.3`

## Python Packaging Notes

- PyPI publishing runs from `.github/workflows/release.yml` using GitHub OIDC trusted publishing.
- Release tags remain `vX.Y.Z`, while Python prereleases map to PEP 440 versions such as `X.Y.Zrc1`.
- Linux is currently published as an `sdist` only. macOS Intel, macOS Apple Silicon, and Windows ship prebuilt wheels.
